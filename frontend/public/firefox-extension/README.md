# 🦊 Firefox 扩展安装指南

## 📥 超简单安装步骤

### 1. 下载并解压
- 下载 `firefox-extension.zip`
- 解压到任意文件夹

### 2. 在Firefox中安装
1. 打开Firefox浏览器
2. 在地址栏输入：`about:debugging`
3. 点击左侧的"此 Firefox"
4. 点击"临时载入附加组件"
5. 选择解压文件夹中的 `manifest.json` 文件
6. 完成！

## 🎯 使用方法

1. 访问OJ平台（洛谷、LeetCode等）并登录
2. 页面右上角会出现橙色🦊按钮
3. 点击按钮获取Token
4. Token自动复制到剪贴板

## ⚠️ 注意事项

- Firefox重启后需要重新加载扩展
- 这是Firefox的安全机制，属于正常现象
- 重新加载只需重复步骤2即可
